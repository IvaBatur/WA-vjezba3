## Pokretanje projekta


### Kloniranje repozitorija

Prvo treba klonirati projekt:
```bash
git clone https://github.com/IvaBatur/WA-vjezba3.git 
cd WA-vjezba3
```

### Backend
```bash
cd pizza-express
npm install
node index.js
```

Pizza poslužitelj sluša na portu 3000

Ostaviti ovaj terminal otvoren! Backend mora raditi cijelo vrijeme.

### Frontend

Otvoriti novi terminal:
```bash
cd pizza-vue/vue-project
npm install
npm run dev
```

 `http://localhost:5173`
